## Sampling Rate

- **After every task commit:** Run focused unit tests for the changed module (`pytest tests/test_<module>.py -x`)
- **After every plan wave:** Run full suite (`pytest tests/ -q`)
- **Before `/gsd-verify-work`:** Full suite must be green AND manual UAT items signed off
- **Max feedback latency:** ~5 seconds for automated; manual UAT requires Flame restart per `memory/flame_module_reload.md`
