## From forge_flame/fbx_io.py:51-70 (THE duck-type pattern to mirror verbatim)
```python
def iter_keyframable_cameras(action):
    """Return Action child nodes that are *keyframable cameras*.

    Duck-typing rule: a keyframable camera has position, rotation, fov,
    and focal PyAttributes. Perspective has these too but is explicitly
    filtered (see memory/flame_perspective_camera.md — non-keyframable
    viewport camera; baking it is a known crash path).
    """
    out = []
    for n in action.nodes:
        if not all(hasattr(n, attr) for attr in ("position", "rotation", "fov", "focal")):
            continue
        if n.name.get_value() == "Perspective":
            continue
        out.append(n)
    return out
```
