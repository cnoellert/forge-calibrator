# Phase 04.1 Plan 01: Stereo-rig filter in _FLAME_SIDE_TEMPLATE Summary

**One-liner:** Duck-type camera filter in `_FLAME_SIDE_TEMPLATE` drops FBX-internal nodes and stereo-rig siblings (`*_left`, `*_right`) from the Send-to-Flame popup's created-list using the same `hasattr(position+rotation+fov+focal)` + Perspective-name exclusion predicate as `iter_keyframable_cameras`.
