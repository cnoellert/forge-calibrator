<bridge_contract>
forge-bridge `/exec` (POST http://127.0.0.1:9999/exec) — current contract:

  Payload:   {"code": "<python source>"}
  Headers:   Content-Type: application/json
  Response:  {"result": "<repr of last expression>", "stdout": "...", "stderr": "...", "error": null|str, "traceback": null|str}

Hard rules (violating any of these crashes Flame and ends the spike):
  R1. REPL semantics — the last bare expression's repr() becomes `result`. Setting `_result = ...` does NOT propagate. End every code body with a bare expression that names what you want returned.
  R2. One probe per /exec call — never bundle multiple introspections in one request. If it crashes, you can't tell which call did it.
  R3. NO QDialog / QWidget / QApplication.exec() reachable from any /exec invocation. The bridge runs off-main-thread and any modal Qt UI from a worker thread is a hard SIGSEGV on macOS. PyExporter().export() does NOT open a dialog (verified in 260430-ddi Probe 3 docstring), but if anything you call surfaces UI, STOP.
  R4. NO iteration over many (Action, Camera) FBX export pairs in one /exec — bridge SIGSEGVs on selection thrashing. Not relevant here, but stay alert.
  R5. If a probe SIGSEGVs (no envelope returned, curl times out / connection refused), STOP, save partial findings to PROBE.md immediately, ask the user to restart Flame, and document the crash itself as the finding. Do NOT retry the same call without changing strategy. After two consecutive crashes, abort the spike with partial findings written down.
  R6. Any state-mutating bridge call (export, create_node, save) must run inside a throwaway batch group named `spike_260430_e5y`. Create it with `flame.batch.create_batch_group(name="spike_260430_e5y")` if it doesn't exist; switch to it before any mutation. The user's open `soc_0090` batch must NEVER be touched.

  R7 (e5y-specific). DO NOT call `flame.PyExporter().get_presets_base_dir(PresetVisibility.Autodesk)` — it SIGSEGV'd the bridge in 260430-ddi step 3.4 and likely will again. Discover the preset path via filesystem search instead (Bash, not bridge).

Latency on the test plate: <dt_seconds>s for <WxH>@<bit_depth>.

import os
import shutil
import tempfile
from typing import Optional, Tuple

_SNAPSHOT_PRESET = "<discovered absolute path>"  # PNG-8bit Image_Sequence preset

def extract_frame_via_export(
    clip, target_frame: Optional[int] = None,

    """Extract one frame via PyExporter snapshot. Returns (raw_bytes, w, h, bit_depth)."""
    import flame

        return None
    try:

        return None

    except Exception:
        return None
    finally:
